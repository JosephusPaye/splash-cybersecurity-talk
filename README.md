Open `index.html` in a browser to view the slides.

To export as PDF, open `index.html?print-pdf` in Chrome or Edge and print to PDF. By default, fragments are collapsed so each slide is a single page. To print each fragment step as a separate page, add `&separate-fragments` to the URL: `index.html?print-pdf&separate-fragments`.
